## Me aparece el error "Certificado no emitido por AC de confianza"

Este error puede ocurrir por 2 motivos:

- Estás utilizando un certificado de producción en un entorno de pruebas. O por el contrario, un certificado de pruebas en un entorno de producción.
- Si creaste un certificado para representar a un tercero, deberás asegurarte de delegar correctamente el CUIT y el computador fiscal, asi mismo asegurarte que el punto de venta esté habilitado para web services

## Me aparece el error "No se pudo replicar el total de la orden"

Este error sucede cuando el plugin no puede entender los totales de la órden. En este caso deberás contactarte con nosotros para solucionarlo.
